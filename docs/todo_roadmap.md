## 🛠️ 작업 상세 기록

### 1. AI 라디오 기능 구현 및 최적화 (P0)
*   **배경**: 사용자의 사연을 바탕으로 한 AI 도슨트/라디오 기능을 핵심 기능으로 구축.
*   **요구사항 (업데이트됨)**:
    *   **DB 연동**: `audio_files` 테이블의 `radio_url`, `radio_url_status` 필드를 활용한 데이터 정합성 유지.
    *   **R2 구조 적용**: `media/radio/[nfc_id]_tts.mp3` 경로 규칙에 맞춰 파일 생성 및 저장.
    *   **생성 프로세스**: 사연(story) -> 스크립트(radio_script) -> TTS 생성 -> R2 업로드 -> DB 상태 업데이트(`TRUE`).
    *   **플레이어 연동**: 메인 음악과 라디오 음원을 유동적으로 전환하며 재생하는 로직 구현.

### 2. 반응형 설계 개선 (P1)
*   **배경**: 현재 `globals.css`가 440px 고정폭으로 설정되어 있어 데스크탑에서 공간 활용도가 낮음.
*   **요구사항**:
    *   `@media (min-width: 768px)` 환경 대응 및 OBS 가로 모드 최적화.

### 3. 플레이어 전역 개편 (P2)
*   **배경**: 단순 버튼 클릭형 UI를 전문 음악 플레이어 인터페이스로 교체.
*   **요구사항**:
    *   앨범 아트워크, 프로그레스 바, Glassmorphism 디자인 적용.
