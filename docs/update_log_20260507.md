# 🧀 JerryKeyring 업데이트 내역 (2026.05.07)

## 1. 플레이어 코어 엔진 고도화
- **통합 재생 시스템 (`startPlayback`) 도입**: 모든 재생 경로(목록 클릭, 자동 다음 곡, 랜덤 재생)를 하나의 함수로 단일화.
- **백그라운드 안정성 확보**: AI 라디오와 본 음원 재생 시 **단일 플레이어 객체**를 재사용하여 iOS 백그라운드 끊김 문제 해결.
- **순차 재생 로직 적용**: `AI 라디오 ON` 설정 시 "라디오 사연 -> 본 음악" 자동 연속 재생.

## 2. 메인 UI 디자인 리뉴얼 (Premium Glassmorphism)
- **헤더 레이아웃 개편**: 
  - 좌측: 로고
  - 우측: [음파 아이콘] + "음원 생성하기" 필(Pill) 버튼, 설정 기어 아이콘.
- **큐레이션 카드 시스템**:
  - 가로 스크롤 방식 적용 (1.5개 카드 노출로 드래그 유도).
  - 정사각형 라운드 카드 디자인 (카테고리명 좌측 하단 배치).
  - 주황(#ff914d) & 검정 테마의 글래스모피즘 적용.
- **하단 목업**: "새로운 기능 업데이트 예정" 안내 박스 배치.

## 3. 사용자 경험(UX) 및 설정 개선
- **지능형 화면 유지**: 메인 UI 탐색 중 자동 곡 전환 시 플레이어 화면 강제 이동 방지.
- **풀스크린 설정**: 환경설정 UI를 풀스크린으로 확장 및 디자인 리뉴얼.
- **메타데이터 연동**: 잠금 화면 아티스트명을 수파베이스 사용자 이름(`userName`)으로 실시간 매핑.

## 4. 관리자(Admin) 시스템 업데이트
- **내비게이션 확장**: 사이드바에 '콘텐츠 큐레이팅' 메뉴 추가.
- **관리 페이지 틀 구성**: `/admin/curation` 페이지 신규 생성.
